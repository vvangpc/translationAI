喵喵翻译

轻量翻译扩展 · 划词气泡 / 右键菜单

翻译引擎

⚠ Google 走非官方免费端点，无需 API key。质量低于 LLM，且高并发可能临时限流（429）。
常见示例： DeepSeek https://api.deepseek.com/v1 · 智谱 https://open.bigmodel.cn/api/paas/v4 · OpenAI https://api.openai.com/v1
对不支持思考模式的模型/服务商无影响。
☁️ 配置随 Chrome 登录的 Google 账号自动同步到所有设备。API key 由 Chrome 加密传输与存储； 若关闭 Chrome 同步则等同于本地存储。

翻译行为

留空使用默认。仅 LLM 引擎生效。
默认 100。划词翻译气泡显示后，页面滚动累计超过此距离会自动关闭气泡。 设为 0 表示禁用滚动关闭（仍保留 ESC / 点击空白处 / X 三种关闭方式）。

域名黑名单

子域自动匹配：填 example.com 会同时禁用 www.example.com。 右键菜单 / 快捷键 / popup 操作均不受影响 —— 仅屏蔽划词时自动弹出的图标。 也可以在 popup 里一键加入/移出当前域名。

配置备份

导出当前所有设置（含黑名单、API Key 明文）到 .json；导入会覆盖现有配置。 不含翻译缓存。文件请妥善保管，不要上传到公开平台。

快捷键

默认 Alt+T 翻译选中文本。
如需修改：在 Chrome 地址栏打开 chrome://extensions/shortcuts。
